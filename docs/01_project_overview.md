## 1. 프로젝트 개요

Manufacturing AI은 설비 센서 데이터와 제조 KPI를 활용하여 설비 상태를 분석하고, 위험 설비를 탐지하며, 정비 우선순위를 제안하는 AI 기반 제조 분석 시스템입니다.

본 프로젝트는 공개 설비 센서 데이터셋(AI4I Predictive Maintenance Dataset)을 기반으로 진행되었으며,
실제 제조 현장에서 활용되는 OEE, 가동률, 품질률 등의 운영 KPI를 참고하여
프로젝트 목적에 맞는 KPI를 직접 설계하였습니다.

설계된 KPI와 센서 데이터를 통합하여 제조 AI 분석 환경을 구축하였습니다.

---

## 2. 프로젝트 배경

제조 현장에서는 다양한 센서 데이터를 수집하고 있으나, 이를 종합적으로 분석하여 설비 상태를 판단하고 정비 우선순위를 결정하는 데 많은 시간이 소요됩니다.

특히 다음과 같은 문제가 발생합니다.

- 설비 고장을 사전에 예측하기 어렵다.
- 위험 설비를 신속하게 식별하기 어렵다.
- 생산성 저하 원인을 분석하기 어렵다.
- 여러 KPI를 동시에 판단하기 어렵다.

본 프로젝트는 이러한 문제를 해결하기 위해 설비 센서 데이터와 제조 KPI를 통합 분석하는 AI 시스템을 구현하는 것을 목표로 합니다.

---

## 3. 프로젝트 목표

- 설비 상태 분석
- 설비 고장 위험 탐지
- OEE 기반 운영 효율 분석
- 정비 우선순위 추천
- 자연어 기반 AI 분석 기능 제공
