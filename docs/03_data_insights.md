## 1. 데이터 개요

- 전체 데이터 수 : 10,000건
- 설비 수 : 10,000대
- 생산 라인 : Line 101, Line 102, Line 201
- 분석 대상 : 설비 센서 데이터 및 제조 KPI

---

## 2. 주요 KPI 현황

| KPI                      | 의미          |
| ------------------------ | ------------- |
| (\*)Production_Qty       | 생산량        |
| (\*)Availability         | 가동률        |
| (\*)Quality_Rate         | 품질률        |
| (\*)OEE                  | 설비종합효율  |
| (\*)Risk_Level           | 위험도        |
| (\*)Maintenance_Priority | 정비 우선순위 |

---

## 3. 주요 분석 가설

### 가설 1

공구 마모가 증가할수록 생산량과 OEE가 감소한다.

---

### 가설 2

토크 이상이 발생할수록 설비 위험도가 증가한다.

---

### 가설 3

가동률이 낮은 설비는 OEE가 낮다.

---

### 가설 4

High Risk 설비는 높은 불량률과 낮은 생산량을 가진다.

---

## 4. 분석 질문

- OEE가 가장 낮은 설비는 무엇인가?
- 어떤 라인이 가장 낮은 OEE를 가지는가?

- 생산량 감소의 원인은 무엇인가?
